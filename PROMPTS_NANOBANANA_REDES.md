# Prompts NanoBanana para Redes Sociales (Roger Lab)

## Version 1: Corporativo
Crea una imagen promocional para redes sociales de Roger Lab (1080x1350), con estilo corporativo premium, limpio y profesional.

Objetivo: transmitir perfil profesional y empleabilidad para conseguir una oportunidad laboral en empresa.

Texto principal:
- Titular: "Desarrollador Web Full Stack"
- Subtitular: "Construyo aplicaciones web claras, mantenibles y orientadas a resultados"
- Credibilidad: "Laravel · JavaScript · React · Node.js"
- CTA: "Disponible para unirme a tu equipo"

Direccion visual:
- Composicion ordenada con jerarquia clara: titular arriba, visual central (proyectos y codigo), cierre abajo con disponibilidad laboral.
- Diseno sobrio, elegante, con mucho aire y equilibrio.
- Incluir recursos visuales relacionados con rendimiento tecnico, trabajo en equipo y calidad de codigo.
- Evitar estilo recargado o efectos excesivos.

Paleta sugerida:
- Fondo principal: #222222
- Acentos: #999999
- Texto claro: #DDDDDD
- Variante light opcional: fondo #ECE8DF y texto #1F1F1F

Requisitos:
- Alto contraste, legibilidad excelente en movil.
- Margenes seguros para feed.
- Sin marcas de agua, sin apariencia de stock generico.

---

## Version 2: Creativo / Atrevido
Disena una pieza para redes sociales (1080x1350) para Roger Lab con un enfoque visual mas creativo, audaz y memorable, manteniendo profesionalidad.

Objetivo: destacar en feed y reforzar marca personal para atraer la atencion de reclutadores y equipos de tecnologia.

Texto:
- Titular: "Codigo limpio, impacto real"
- Subtitular: "Desarrollador web preparado para aportar desde el primer sprint"
- Soporte: "Frontend · Backend · Arquitectura mantenible"
- CTA: "Abierto a nuevas oportunidades"

Direccion visual:
- Composicion dinamica con diagonales o bloques asimetricos.
- Fondo con textura sutil o gradiente tecnico (sin caer en look generico).
- Elementos visuales: interfaz web, fragmentos de codigo limpios, tarjetas de proyecto.
- Sensacion: energia, criterio tecnico y proyeccion profesional.

Paleta sugerida:
- Base oscura #222222
- Contraste #DDDDDD
- Acento neutral #999999
- Permitir un unico acento adicional discreto para foco visual.

Requisitos:
- Mantener lectura clara del titular y CTA.
- No sobrecargar con texto.
- Debe funcionar bien en mobile first.

---

## Version 3: Performance Ads (Meta/IG)
Genera una creatividad para anuncio de rendimiento en redes (1080x1350), optimizada para visibilidad profesional y captacion de oportunidades laborales.

Objetivo: conseguir contactos de recruiters, hiring managers y empresas tecnicas.

Copy de la pieza:
- Hook: "¿Buscas desarrollador web para tu equipo?"
- Beneficio: "Aporto desarrollo full stack, enfoque en calidad y mejora continua"
- Prueba tecnica: "Laravel · JavaScript · React · Node.js"
- CTA fuerte: "Disponible para entrevista"

Direccion visual orientada a performance:
- Foco inmediato en hook + CTA.
- Alto contraste y lectura instantanea en 1-2 segundos.
- Visual central: portfolio tecnico, repositorios, modulos de app y evidencias de buenas practicas.
- Uso de badges cortos: "Clean Code", "Laravel", "JavaScript", "Team Ready".

Paleta sugerida:
- Fondo: #222222
- Texto principal: #DDDDDD
- Texto secundario: #CFCFCF
- Bordes/acento: #999999

Reglas para ads:
- Maximo 3 bloques de texto.
- Sin parrafos largos.
- Zona segura para recortes de interfaz social.
- Estetica moderna y confiable, no clickbait.

---

## Recomendacion de uso rapido
1. Empieza con la Version 1 para posicionamiento profesional en perfil.
2. Usa la Version 2 para alcance y marca personal tecnica.
3. Usa la Version 3 para campanas orientadas a recruiters y hiring managers.
